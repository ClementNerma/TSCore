# Examples

|             Name              |            Description           |        Arguments         |
|-------------------------------|----------------------------------|--------------------------|
| [`fetch-todo`](fetch-todo.ts) | Fetch a todo note from a website | Todo note's ID (integer)

## Running an example

To run an example, simply go to the [`examples/run`](run/) folder and run:

```shell
yarn <example> [<arguments>]

# Example:
yarn fetch-todo 1
```
